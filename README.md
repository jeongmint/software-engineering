# Markdown Syntax 정리

## 제목(Header)

* h1 부터 h6 까지 '#' 기호를 통해 헤더 층위 변경

```markdown
# h1

## h2

### h3

#### h4

##### h5

###### h6

```
==== **결과** ====

# h1

## h2

### h3

#### h4

##### h5

###### h6

============ <br/><br/>

* 마크다운 문법은 한 줄 씩 띄어서 작성하는 것이 원칙이며 제목과 문단 조합을 사용하기 위해서는 다음과 같이 사용

```markdown

# 제목

제목에 해당하는 문단입니다.

```

==== **결과** ====

# 제목

제목에 해당하는 문단입니다.

============ <br/><br/>

* 제목을 # 외에 '=' 혹은 '-' 로 나타낼 수도 있음

```markdown

h1
===============

h2
---------------

### h3
```

* 문단 상에서 엔터를 여러 번 친다고 개행이 되지 않기 때문에 강제 개행을 위해서는 <br/>
<
br/> 태그를 사용

```markdown

## LILAC

오 라일락 꽃이 지는 날 goodbye<br/>
이런 결말이 어울려<br/>
안녕 꽃잎 같은 안녕<br/>

```

==== **결과** ====

## LILAC

오 라일락 꽃이 지는 날 goodbye<br/>
이런 결말이 어울려<br/>
안녕 꽃잎 같은 안녕<br/>

============ <br/><br/>

## 폰트 강조/꾸미기(Emphasis/Decoration)

### - *이탤릭체*

```markdown

이텔릭체는 *Italic* 혹은 _Italic_ 처럼
하나의 '*' 혹은 '_'를 문자 양 옆에 사용

```

### - **볼드체**

```markdown

볼드체는 **Bold** 혹은 __Bold__ 처럼
두 개의 '**' 혹은 '__'를 문자 양 옆에 사용

```

### - <u>밑줄</u>

```markdown

밑줄은 <u>Highlight it!</u> 처럼
tag u를 문자 양 옆에 사용

```

### - ~~취소선~~

```markdown

취소선은 ~~Cancel it!~~ 처럼
두 개의 물결 '~~'을 문자 양 옆에 사용

```

==== **결과** ====

-이텔릭체는 *Italic* 혹은 _Italic_

-볼드체는 **Bold** 혹은 __Bold__

-밑줄은 <u>Highlight it!</u>

-취소선은 ~~Cancel it!~~


============ <br/><br/>
